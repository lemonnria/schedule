## 反思

- 拿到项目应多分析项目内容，比如哪些要固定不动，以及页面的一些 div 的层次，不然后期改动很麻烦 ORZ

- 样式的命明要规范，不然自己可能都看不懂
- 有些地方可以用插件写，但是自己还会应用，网上的代码应该再深入学习而不是不了解一些含义就直接用
